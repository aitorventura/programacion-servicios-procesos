# 🔁 Tema 0: Repaso de Java y programación funcional

!!! note "Tema de repaso — sin RA asociado"
    Este tema no evalúa ningún resultado de aprendizaje del currículo. Repasa contenidos que ya se han visto en el módulo de Programación (relaciones entre clases y colecciones) y añade uno nuevo, la programación funcional con streams, que se va a usar mucho durante el curso para trabajar con datos.

Antes de meternos con procesos, hilos y sockets, conviene tener a mano algunas herramientas de Java: cómo se relacionan las clases entre sí, cómo guardar colecciones de objetos y cómo recorrerlas de forma funcional en lugar de con bucles. Nada de esto es nuevo del todo, pero es la base sobre la que va a apoyarse el resto del módulo.
